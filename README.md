# All-the-News-That-s-Fit-to-Scrape

This app will allow users to scrape an articles from CBS News. Also allows users to leave note. 

Technology Used

* Node.js
* MongoDB 
* Mongoose 
* Express
* Express 
* Axios 
* Cheerio
